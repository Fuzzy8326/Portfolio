Projects

Shopping Cart

A shopping cart application built using HTML, CSS, and JavaScript. It allows users to add or remove items from the cart, adjust quantities, and view the total cost. The design is user-friendly. JavaScript handles dynamic updates,and cart state management, while HTML and CSS provide the structure and styling for a clean interface.

Guessing Game

A fun and interactive game where players guess a random number between 1 and 20. Built with HTML, CSS, and JavaScript, it provides instant feedback, tracks attempts, and celebrates when you guess correctly.This project showcases key JavaScript concepts like conditionals and loops, making it a great example of user-driven interactivity.

Super Calculator

This calculator is built with HTML, CSS, and JavaScript, it goes beyond basic arithmetic by supporting addition, subtraction, multiplication, division, and modulo operations. But that’s not all — this smart calculator can concatenate words and sentences, turning text into seamless combinations. It also features comparison operators to check if values are equal, greater, or smaller, returning true or false instantly. A fun, interactive project that demonstrates JavaScript operators, user input handling, and DOM manipulation in action.

Runaway Rails

An interactive game where you control a train racing down a track using JavaScript functions. Each click on the train increases its speed, but push too hard and you risk a high-speed crash at the end of the track! The challenge? Stop the train in time using the Stop button. Built with HTML, CSS, and JavaScript, this project demonstrates functions, event handling, and dynamic speed control while delivering a fun and fast-paced gameplay experience.

Full Stack Pizza

A dynamic web app for a local pizza shop that started simple but grew to meet customer demands. Originally offering just cheese and pepperoni for local delivery, it now includes multiple pizza options, extended delivery areas, and special birthday deals. Built with HTML, CSS, and JavaScript, this project demonstrates form handling, conditional logic, and dynamic content updates to create a user-friendly ordering experience.

Story Spinner!

An interactive web app that lets players fill in nouns, verbs, adjectives, adverbs, and more to create unique, funny stories. Using HTML, CSS, and JavaScript, the game combines user input with a story template via the string concatenation operator, so every run produces a different result. This project demonstrates dynamic content generation, variables, and DOM manipulation in a fun and creative way.
